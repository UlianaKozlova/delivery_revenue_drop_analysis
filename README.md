# Исследование падения оборота сервиса доставки продуктов

**Цель проекта** – определить причины снижения выручки в апреле 2026 года по сравнению с предыдущими месяцами.

**Данные** – три таблицы за период ноябрь 2025 – апрель 2026:
- Заказы
- Пользовательская воронка
- Маркетинговые коммуникации

**Результаты**:
- Падение оборота вызвано прежде всего снижением конверсии на этапе «Карточка товара → Добавление в корзину».
- Проблема затронула все платформы, что указывает на общий сбой (бэкенд или бизнес-логика).
- Средний чек также статистически значимо снизился.
- Маркетинговая активность сократилась, но это не главная причина.

**Рекомендации** – см. `reports/conclusions.md`.

**Запуск**:
1. Установите зависимости: `pip install -r requirements.txt`
2. Поместите CSV-файлы в папку `data/`.
3. Запустите `notebooks/analysis.ipynb`.
